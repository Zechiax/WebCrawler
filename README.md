## How to run

### Docker Compose

Clone the repository

```bash
git clone https://github.com/Zechiax/WebCrawler.git
```

Go to the project directory

```bash
cd WebCrawler
```

Run the project

```bash
docker compose up
```

Navigate to [localhost:8080](http://localhost:8080), you can change the port in the `docker-compose.yml` file.
